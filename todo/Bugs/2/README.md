### 已反馈但未修复的 bugs

本文档列举一些已反馈但未修复的 bugs。

#### 奥比岛手游一加渠道服登录时一直循环在选择账号上就是登不上去

- **原因**：大概率是正在使用的系统不是一加大陆版系统，奥比岛访问移动服务的优先级低于一加账号（或其它系统自带的账号管理器）。
- **解决方案**：~~问客服，客服说修复下，哎呀别为难人家客服啦~~~使用隐藏应用列表或其它方式让奥比岛（com.leiting.aobi.nearme.gamecenter）检测不到一加账号（com.oneplus.account）的存在，安装一加版的移动服务（com.heytap.htms），在移动服务上登录即可。记得自行核对一下应用包名，可以使用 MT 管理器或其它方式获取。

#### 奥比岛手游一加渠道服无法登录或登录时不显示移动服务的图标

- **原因**：移动服务未能被奥比岛手游识别或移动服务版本太新。
- **解决方案**：安装旧版移动服务，可尝试在 [../src/](../src/) 中查找旧版移动服务安装包。
